# 🛒 Симулятор Супермаркета 3D

## Как получить APK через GitHub (бесплатно, без Android Studio)

### Шаг 1 — Создай аккаунт на GitHub
Зайди на [github.com](https://github.com) и зарегистрируйся (бесплатно).

### Шаг 2 — Создай репозиторий
1. Нажми **New repository**
2. Назови его `supermarket-simulator`
3. Выбери **Public**
4. Нажми **Create repository**

### Шаг 3 — Загрузи файлы
1. Нажми **uploading an existing file**
2. Перетащи ВСЕ файлы из папки проекта
3. Нажми **Commit changes**

### Шаг 4 — Запусти сборку
1. Перейди на вкладку **Actions**
2. Слева выбери **Build Android APK**
3. Нажми **Run workflow** → **Run workflow**
4. Подожди 5-10 минут

### Шаг 5 — Скачай APK
- После завершения нажми на сборку
- Внизу страницы найди **Artifacts**
- Скачай **SupermarketSimulator-APK**
- Внутри архива будет `SupermarketSimulator.apk`

## Управление (ПК)
- **WASD** — движение
- **Shift** — бег
- **Пробел** — прыжок
- **E** — взаимодействие (касса, кровать)
- **Esc** — курсор мыши

## Управление (Телефон)
- **Джойстик** (левая половина экрана) — движение
- **Свайп** (правая половина) — камера
- **⬆ кнопка** — прыжок
- **E кнопка** — взаимодействие
